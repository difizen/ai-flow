# Foo

## Normal

```jsx
import React from 'react';
import { Foo } from '@alipay/ai-workflow';

export default () => <Foo>Hi, bigfish</Foo>;
```

## Large

```jsx
import React from 'react';
import { Foo } from '@alipay/ai-workflow';

export default () => <Foo size="large">Hi, bigfish</Foo>;
```
